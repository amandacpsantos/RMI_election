"# election-java-rmi-1072788" 

## Como utilizar

Vá para a pasta bin do projeto e siga as orientações abaixo:
1. No primeiro prompt de comando, digite ```rmiregistry```
2. No segundo prompt de comando, digite: ```java redes.ElectionServer```
3. No terceiro ou mais prompts de comando para instanciar clientes, digite ```java redes.ElectionClient```

Ao iniciar redes.ElectionClient terão os comandos para seguir a utilização.
